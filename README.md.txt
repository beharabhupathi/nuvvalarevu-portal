# Smart Nuvvalarevu Digital Platform

This project ensures all villagers—especially illiterate residents—receive government benefits through a transparent, scalable system.

## Features
- Google Sheet data depository
- Website for awareness and issue reporting
- Glide App for admin control
- Urgent alerts via SMS/WhatsApp

## Technologies
- HTML, CSS, JavaScript
- Google Sheets + Apps Script
- Glide App
- GitHub Pages or Netlify for hosting

## Admin Roles
- Each street has one admin
- Bhupathi is fallback admin for all streets

## Impact
- Supports illiterate residents
- Improves transparency
- Empowers village with digital tools
